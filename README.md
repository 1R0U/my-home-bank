# my-home-bank
# 🏦 我が家中央銀行 (OurHome Central Bank)

> 家族の「めんどくさい」をマイニング（採掘）に変える、家庭内独立経済圏シミュレーションOS。

「宿題やった？」「お風呂掃除してよ！」という日常の義務感や小競り合いを、家族全員で楽しむ「経済ゲーム」へと昇華させるクローズドなWebアプリケーションです。

---

## 🌟 開発の背景と届けるアウトカム（価値）

人間は「やらなきゃいけない（義務）」と思うと動けませんが、「コインが稼げる（クエスト）」となった瞬間に能動的になります。
本プロジェクトは、外部APIや資金を一切使わず、**「データベースの設計」と「ゲーム性のロジック」だけ**で、家族の行動変容という最大のアウトカムを届けます。

### 🎯 本質的なアウトカム
1. **家事・義務の自発化**: 命令や小言がなくなり、家族が自らタスクを探して動くようになります。
2. **生きた金融教育**: 貯金、利息、ローン、インフレといった社会の仕組みを、家庭内のリアルなやり取りで学べます。
3. **家族の行動ログの資産化**: 「誰が・いつ・何に貢献したか」が数年単位のデータとして蓄積され、家族の成長記録になります。

---

## 🛠 主な機能（コア・ロジック）

外部連携をあえて行わず、家族間の「手動承認」と「数値の増減」だけで完結する堅牢なシステムです。

### 1. タスク・マイニング（通貨の発行）
*   家庭内の「めんどくさいこと」をクエストとして登録。
*   行動を完了し、他の家族がアプリ上で「承認」すると、独自家庭内通貨（例：`$HMC` - Home Coin）がウォレットに即時発行されます。
*   *例：お風呂掃除（50 $HMC）、宿題（30 $HMC）、マイナス行動（片付け忘れ：-20 $HMC）*

### 2. 中央銀行セクター（金利・融資ロジック）
*   **定期預金**: 稼いだ通貨を銀行に預けることで、設定された「週利・月利」に応じて自動で利息が発生。貯金する楽しさを育みます。
*   **家庭内ローン**: 「どうしても今あのゲームが欲しい！」という子供（または大人）に対し、中央銀行が融資。毎月の給与（お手伝い報酬）から天引きされる金利付きの返済計画をシミュレーションします。

### 3. 外貨両替・ストア（リアル報酬との交換）
*   貯まった通貨は、家庭内の「リアルな権利」と交換（購入）できます。
*   *交換アイテム例：「今日の晩御飯リクエスト権」「ゲーム1時間延長券」「パパにマッサージしてもらえる券」「お小遣い（現実の現金）への両替」*
*   家庭内の需給に合わせて、アイテムの価格が変動する「インフレ・デフレ機能」も実装予定。

## 🗄 データベース設計の方向性（アーキテクチャ）

資金ゼロ・外部連携なしで運用するため、シンプルなリレーショナルデータベース（SQLite / PostgreSQLなど）で完結させます。

### 必要な主要テーブル（概念）
*   `Users`: 家族のメンバー情報（ロール：親/子、現在の残高）
*   `Quests`: めんどくさい行動のマスターデータ（報酬額、承認に必要な人数）
*   `QuestLogs`: 誰がいつクエストをクリアし、誰が承認したかの履歴（これが家族の歴史データになる）
*   `StoreItems`: 交換できる景品・権利のマスターデータ
*   `BankAccounts`: 預金口座、ローン残高、金利計算用テーブル

*   **Phase 2 (エコノミーの実装)**: ストア機能（権利との交換）の実装、定期預金の自動利息計算ロジックの追加。
